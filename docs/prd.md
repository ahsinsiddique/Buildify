# Product Requirements

## Vision

Replace spreadsheets and disconnected workflows with a centralized system for property sales, construction execution, finance, inventory, and CRM.

## Users

- `admin`: full platform access
- `manager`: projects, workers, inventory, CRM
- `accountant`: invoices, payments, expenses, reporting
- `worker`: assigned tasks, attendance, limited project updates

## MVP Modules

### Property Management

- Create and update plots, houses, and inventory units
- Track status, location, price, and buyer association
- Store ownership and reference documents

### Construction Projects

- Create projects linked to properties
- Track budget, deadlines, and progress
- Record daily logs, labor, and material usage

### Workforce

- Manage worker records and roles
- Track assignments, attendance, and wage data

### Finance

- Record expenses by category and project
- Create invoices and track payments
- Surface project profitability and cash flow summaries

### Materials

- Maintain material catalog and vendor references
- Track purchases and usage per project

### CRM

- Manage clients, consultation notes, and follow-ups
- Associate deals to properties and projects

### Dashboard

- Total expenses, invoiced amount, outstanding balance
- Project-level cost and progress summaries
- Daily spending trends
- Worker and materials cost breakdown

## Non-Functional Requirements

- Role-based access control on all business actions
- Mobile-friendly interface for site usage
- Audit-friendly timestamps on critical records
- Modular architecture that supports future AI insights and alerts
